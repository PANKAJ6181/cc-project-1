# personal portfolio website

This is a simple and clean Personal Portfolio Website built using HTML and little bit of CSS.

It showcases basic details such as About Me, Skills, Experience, Projects, Education, and Contact information.

## Features

- Sections included:
  - About Me
  - Skills
  - Experience
  - Projects
  - Education
  - Contact Details
- Fully responsive basic structure
- Beginner-friendly HTML project
